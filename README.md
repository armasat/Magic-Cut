# Magic-Cut
Magic Cut is an auto-clicker made in BASH

# Dependencies
  - seq
  - shuf
  - head
  - xdotool
  
# Install
Download or Clone the repo to an empty scratch directory and make the shell scripts executable
```shell
sudo chmod +x *.sh
```

# Usage
Left-Click every 1 to 20 seconds (randomly chosen interval)
```sh
sh magicCut.sh 1 20 
```
